# NEXT Horizon

NEXT Horizon is a multi-page static website for a creator-focused brand. It includes a landing page, a Labs page, a product page, and a sign-up page, all connected through a shared navigation, consistent styling, and a small JavaScript menu toggle for mobile layouts.

## Pages

- `index.html` - main landing page with hero, services, and footer sections
- `tech.html` - Labs page highlighting creator tools and workflows
- `products.html` - product showcase page for the featured creator kit
- `signup.html` - sign-up form page for new users

## Built With

- HTML5
- CSS3
- Vanilla JavaScript
- Google Fonts
- Font Awesome

## Features

- Responsive navigation with mobile menu toggle
- Shared branding and layout across all pages
- Product-focused marketing content
- Starter sign-up form UI
- External font and icon integration

## Getting Started

This project does not require a build step or package installation.

1. Clone or download the project.
2. Open `index.html` directly in your browser.

If you want a local development server, you can use the VS Code Live Server extension or run a simple static server from this folder.

## Notes

- The sign-up form is currently frontend-only and does not submit data to a backend.
- The mobile navigation behavior is handled in `app.js`.
- Styling for all pages is centralized in `styles.css`.
